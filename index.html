Введение

Создание сайта с нуля это очень трудоемкий процесс над которым трудится не один специалист (дизайнер, верстальщик, программист ну и конечно объединяет весь процесс менеджер проекта). После написание сайта он должен быть выложен в интернет сеть (хостинг) чтобы доступ к нему был постоянен из любых точек мира при этом у сайта должно быть запоминающееся доменное имя. Ну и на этом работа над сайтом не останавливается т.к. на него нужно приводить посетителей.

Как запустить сайт без особых знаний

- Если нужен сайт и вы хотите его запустить самостоятельно без знаний веб-программирования, тогда вам необходим хостинг который поддерживает PHP и MySQL и на него необходимо установить CMS с понравившемся шаблоном.
- Так что даже без знаний веб-программирования можно вполне и обойтись :-)
- Установив CMS – вы уже получаете готовый сайт со всеми современными свойствами и функциями.
- CMS (система управления сайтом) или её еще называют "Движок". Это универсальное средство для создания своего сайта, которое поможет вам в управлении вашим сайтом, добавлением страниц, картинок, товаров и т.п.
- CMS использует языки программирования и автоматическую инсталяцию после выкладывания её на хостинг. Для инсталяции необходимо просто следовать инструкциям при первом заходе на сайт по вашему доменному имени. Получается что знать языки веб-программирования – совсем не обязательно для запуска сайта с готовыми шаблонами.
- Шаблон - это то ка выглядит ваш сайт (Шапка, меню, подвал и другие его элементы)

Вопросы для написания сайта

  • Для чего вам нужен сайт?
  • Какая информация будет на сайте (текст, картинки ...)?
  • Какое название и тематика сайта для поиска в интернете?
  • Какое оформление будет у сайта (расположение, сочетание цветов ...)?
  • Какие эмоции должен вызывать сайт у посетителей?
  • Какое количество посетителей будет на сайте?
  • Какой будет функционал у сайта (инфомация, блог, магазин и т.п.)?

Инструменты для написания сайта

Для редактирования графических изображений нам понадобится графический редактор Photoshop, несмотря на то что этот инструмент больше нужен дизайнеру, в процессе написания сайта он обязательно пригодится и верстальщику, который преобразует созданный макет дизайнером в язык понятный для отображения сайта в браузере.

Вот мы и подошли до самых главных инструментов для разработки сайтов.

Notepad++ - очень простой текстовый редактор который подходит для профессионалов и начинающих, он показывает текстовую подсветку удобную для написания кода разметки сайта.

Brackets - Профессиональный инструмен который заслуживает большого внимания для тех кто хочет писать сайты профессионально.

Какие языки необходимо знать для верстки страницы

HTML (от англ. HyperText Markup Language — «язык гипертекстовой разметки») — стандартизированный язык разметки документов во Всемирной паутине.

CSS (от англ. Cascading Style Sheets — каскадные таблицы стилей) — формальный язык описания внешнего вида документа, написанного с использованием языка разметки.
