Введение

Создание сайта с нуля - это очень трудоемкий процесс над которым трудится не один специалист (дизайнер, верстальщик, программист ну и конечно объединяет весь процесс менеджер проекта). После написание сайта он должен быть выложен в интернет сеть (хостинг) чтобы доступ к нему был постоянен из любых точек мира при этом у сайта должно быть запоминающееся доменное имя. Ну и на этом работа над сайтом не останавливается т.к. на него нужно приводить посетителей.

Как запустить сайт без особых знаний

Если нужен сайт и вы хотите его запустить самостоятельно без знаний веб-программирования, тогда вам необходим хостинг который поддерживает PHP и MySQL и на него необходимо установить CMS с понравившимся шаблоном.

Установив CMS – вы уже получаете готовый сайт со всеми современными свойствами и функциями.

CMS (система управления сайтом) или её еще называют "Движок". Это универсальное средство для создания своего многостраничного сайта, которое поможет вам в управлении вашим сайтом, добавлением страниц, картинок, товаров и т.п.

CMS использует автоматическую инсталяцию после выкладывания её на хостинг. Для инсталяции необходимо просто следовать инструкциям при первом заходе на сайт по вашему доменному имени. Получается что знать языки веб-программирования – совсем не обязательно для запуска сайта с готовым шаблоном.

Вопросы для написания сайта

  • Для чего вам нужен сайт?
  • Какая информация будет на сайте (текст, картинки ...)?
  • Какое название и тематика сайта для поиска в интернете?
  • Какое оформление будет у сайта (расположение, сочетание цветов ...)?
  • Какие эмоции должен вызывать сайт у посетителей?
  • Какое количество посетителей будет на сайте?
  • Какой будет функционал у сайта (инфомация, блог, магазин и т.п.)?

Какие языки необходимо знать для верстки страницы

"HTML5"

HTML (от англ. HyperText Markup Language — «язык гипертекстовой разметки») — стандартизированный язык разметки документов во Всемирной паутине.

"CSS3"

CSS (от англ. Cascading Style Sheets — каскадные таблицы стилей) — формальный язык описания внешнего вида документа, написанного с использованием языка разметки.

"JavaScript"

JavaScript обычно находит широкое применение в браузерах как язык сценариев для придания интерактивности веб-страницам.

Javascript не требуется компилировать, он подключается к HTML-странице и воспринимается браузером как есть. С помощью Javascript можно изменять страницу, писать на ней текст, добавлять и удалять теги, менять стили элементов с любой длительностью а не сразу как это делает CSS. Javascript может реагировать на события когда что-нибудь случится (клик мыши, окончание загрузки страницы) и реагировать на это выполнением функции.

PHP, MySQL

Если вы хотите иметь многостраничный или многопользовательский сайт с формами заказа или формой обратной связи, которые обрабатываются на стороне сервера то вам необходимо будет изучить еще такие языки как PHP и MySQL

PHP — это язык программирования, выполняемый на стороне сервера. В отличие от языка JavaScript PHP не зависит от программного обеспечения клиента и поэтому будет выполнен в любом случае.

MySQL — это система управления реляционными базами данных.
Сервер MySQL позволяет эффективно работать с данными и обеспечивает быстрый доступ к данным одновременно нескольким пользователям. При этом доступ к данным предоставляется только пользователям, имеющим на это право.

ФРЕЙМВОРК

Для того чтобы упростить работу по созданию сайта необходимо познакомиться со следующими библиотеками (фреймворками)

Bootstrap — это самый популярный в мире HTML, CSS и JS фреймворк для создания адаптивных и мобильных web-проектов.

jQuery — это быстрая, небольшая и богатая возможностями JavaScript библиотека. Она позволяет очень просто делать такие вещи как: обход элементов или манипуляция элементами HTML документа или AJAX запросы.
